# Fundamentos de Desenvolvimento Web

Repositório destinado ao armazenamento de arquivos, materiais e atividades referentes do módulo 'Fundamentos de Desenvolvimento Web' do curso técnico em Desenvolvimento de Sistemas da Escola Proz Educação.

## 📌 Sobre o Módulo

Este repositório contém todos os recursos utilizados durante o módulo, incluindo:

- Anotações
- Exercícios
- Projetos
- Slides
- Materiais complementares

## 📌 Objetivo

Centralizar os materiais do módulo para facilitar:

- Organização do conteúdo
- Consulta futura
- Versionamento dos arquivos
- Compartilhamento com colegas

## 📌 Como Usar

Clone o repositório:

```bash
git clone https://github.com/edersontec/fundamentos-desenvolvimento-web.git
cd fundamentos-desenvolvimento-web
```
